ADB Speed-Safety — Street View Map Assets

This site hosts the Street View thumbnails and dataset that power the interactive kepler.gl speed-safety map.

  • kepler_svi.csv — map dataset (lat/lon, speed fields, and image URLs)
  • thumbs/S####.jpg — labelled Street View image per segment, e.g. S0000.jpg
  • KEPLER_README.md — how to load it in kepler.gl
